Live Code IDE 배포본
======================

이 폴더는 사용자가 바로 실행해서 사용할 수 있도록 준비된 **배포용 빌드(dist)** 를 담고 있습니다.

구성
----
- `Live Code IDE.jar`  
  - fat JAR 실행 파일입니다. Java 21 이상이 설치된 환경에서 바로 실행할 수 있습니다.
- `plugins/`  
  - TM4E / ANTLR 기반 문법 하이라이팅, 자동완성, 심볼 분석에 필요한 플러그인과 설정 파일이 들어 있습니다.

실행 방법 (Windows, Mac)
-------------------
1. Java 21 이상 JDK 또는 JRE가 설치되어 있어야 합니다.
2. 이 `dist` 폴더 전체를 원하는 위치에 압축 해제합니다.
3. `Live Code IDE.jar` 를 더블 클릭해서 실행합니다.
   - 더블 클릭이 안 되면, 콘솔에서 다음과 같이 실행할 수 있습니다.
     ```bash
     java -jar "Live Code IDE.jar"
     ```

주의 사항
---------
- `Live Code IDE.jar` 와 `plugins` 폴더는 **같은 디렉터리 안에** 있어야 합니다.  
  (플러그인 폴더를 찾지 못하면 문법 하이라이팅과 자동완성 기능이 동작하지 않을 수 있습니다.)
- 이 배포본은 개발용/테스트용으로 제공되며, 소스 코드는 `main` / `server` 브랜치에서 관리됩니다.
